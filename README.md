## Getting Started
Clone this repo
create a virtual environment
install the requirements

### Prerequisites
python 3.6+, python virtual environment

### Usage

cd pidetectivevirtualenv
source bin/activate
cd ~/source/PI-Detective
python3 -m src --ip <ip address> --inet 0
